# Operators in C#

## Concept Explanation
Operators are special symbols that tell the compiler to perform an action on one or more values.

Common categories:
- Arithmetic operators: `+`, `-`, `*`, `/`, `%`
- Comparison operators: `==`, `!=`, `>`, `<`
- Logical operators: `&&`, `||`, `!`
- Assignment operators: `=`, `+=`, `-=`

## Real-World Analogy
Think of operators like action buttons on a calculator or decision switches in a machine.

## Syntax
```csharp
int sum = a + b;
bool result = a > b;
bool check = true && false;
```

## Step-by-Step Explanation
1. Arithmetic operators calculate values.
2. Comparison operators return `true` or `false`.
3. Logical operators combine boolean conditions.
4. Assignment operators store results in variables.

## Best Practices
- Use parentheses when expressions are complex.
- Keep examples small and readable.
- Learn operator precedence gradually.

## Common Mistakes
- Confusing `=` with `==`.
- Forgetting integer division rules.
- Writing complex conditions without parentheses.
