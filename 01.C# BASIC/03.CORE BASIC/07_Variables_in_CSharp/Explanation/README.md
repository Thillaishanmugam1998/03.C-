# Variables in C#

## Concept Explanation
A variable is a named memory location used to store data. Each variable has a data type and a name. The stored value can be updated during program execution.

## Real-World Analogy
Think of a variable like a labeled container. The label is the variable name, and the item inside is the value.

## Syntax
```csharp
int age = 20;
string name = "Arun";
age = 21;
```

## Step-by-Step Explanation
1. `int age = 20;` creates a variable named `age`.
2. `string name = "Arun";` creates a text variable.
3. `age = 21;` updates the value stored in the variable.

## Best Practices
- Use meaningful variable names.
- Match the variable name with its purpose.
- Initialize variables before using them.

## Common Mistakes
- Using unclear names like `a` or `x` for important values.
- Forgetting to assign a value before reading it.
- Using the wrong data type.
