# Continue Statement

## Concept Explanation
The `continue` statement skips the rest of the current loop iteration and moves to the next iteration.

## Real-World Analogy
Think of `continue` like skipping one question in an exam and moving to the next question.

## Syntax
```csharp
if (condition)
{
    continue;
}
```

## Step-by-Step Explanation
1. The loop starts.
2. A condition is checked.
3. If true, the current iteration is skipped.
4. The loop continues with the next iteration.

## Best Practices
- Use `continue` to skip specific cases clearly.
- Keep the condition simple.

## Common Mistakes
- Using `continue` too often and reducing readability.
- Forgetting how it affects loop flow.
