# User-Defined Functions

## Concept Explanation
User-defined functions are methods created by developers rather than built-in library methods.

## Real-World Analogy
Think of a user-defined function like creating your own shortcut command for a repeated task.

## Syntax
```csharp
private static int Add(int a, int b)
{
    return a + b;
}
```

## Step-by-Step Explanation
1. Define the function name and return type.
2. Add parameters if needed.
3. Write the function body.
4. Call the function from `Main`.

## Best Practices
- Use clear names.
- Keep parameters meaningful.
- Return results when appropriate.

## Common Mistakes
- Mismatching return types.
- Forgetting to pass required arguments.
