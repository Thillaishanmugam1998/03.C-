# String in C#

## Concept Explanation
A `string` stores text as a sequence of characters. Strings support many useful operations like concatenation, searching, and formatting.

## Real-World Analogy
Think of a string like a sentence written on paper.

## Syntax
```csharp
string name = "Arun";
Console.WriteLine(name);
```

## Step-by-Step Explanation
1. A string variable is declared.
2. Text is assigned using double quotes.
3. String methods like `ToUpper()` can be used.

## Best Practices
- Use meaningful string variable names.
- Prefer interpolation for readable output.

## Common Mistakes
- Using single quotes for strings.
- Forgetting strings are immutable.
