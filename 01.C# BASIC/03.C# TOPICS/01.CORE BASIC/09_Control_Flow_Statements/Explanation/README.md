# Control Flow Statements

## Concept Explanation
Control flow statements determine how a program moves from one line of code to the next. They allow the program to make decisions, repeat tasks, and skip or stop certain parts.

Common control flow statements include:
- `if`
- `if-else`
- `switch`
- `for`
- `while`
- `do-while`
- `break`
- `continue`

## Real-World Analogy
Think of control flow like traffic rules. A signal tells you whether to stop, move, or turn. In the same way, control flow guides the execution of code.

## Syntax
```csharp
if (condition)
{
    // decision
}

for (int i = 0; i < 3; i++)
{
    // repetition
}
```

## Step-by-Step Explanation
1. Decision statements choose one path.
2. Loop statements repeat a block.
3. Jump statements like `break` and `continue` alter the normal flow.

## Best Practices
- Keep conditions simple and readable.
- Use indentation to show program flow clearly.
- Pick the control statement that best matches the problem.

## Common Mistakes
- Writing conditions that are hard to understand.
- Forgetting loop termination logic.
- Misusing `break` or `continue`.
