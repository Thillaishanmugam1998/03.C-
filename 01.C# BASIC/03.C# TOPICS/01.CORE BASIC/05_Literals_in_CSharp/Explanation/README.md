# Literals in C#

## Concept Explanation
Literals in C# are fixed values written directly in the source code. They represent the actual data assigned to variables or used in expressions.

Common literal types include:
- Integer literals like `10`
- Floating-point literals like `12.5`
- Character literals like `'A'`
- String literals like `"Hello"`
- Boolean literals like `true` and `false`

## Real-World Analogy
Think of literals as ready-made labels already printed on boxes. You do not calculate them at runtime. You write them directly in the code.

## Syntax
```csharp
int count = 10;
double price = 25.5;
char letter = 'C';
string text = "Hello";
bool isOpen = true;
```

## Step-by-Step Explanation
1. `10` is an integer literal.
2. `25.5` is a floating-point literal.
3. `'C'` is a character literal.
4. `"Hello"` is a string literal.
5. `true` is a boolean literal.

## Best Practices
- Match the literal type with the variable type.
- Use clear and meaningful values in beginner examples.
- Use single quotes for `char` and double quotes for `string`.

## Common Mistakes
- Using double quotes for characters.
- Using single quotes for strings.
- Assigning a decimal literal to an `int`.
