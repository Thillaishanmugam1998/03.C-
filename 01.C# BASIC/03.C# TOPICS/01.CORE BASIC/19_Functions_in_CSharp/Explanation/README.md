# Functions in C#

## Concept Explanation
Functions are named blocks of code used to perform a specific task. In C#, functions are usually called methods.

## Real-World Analogy
Think of a function like a kitchen appliance button. Pressing the same button performs the same task each time.

## Syntax
```csharp
private static void SayHello()
{
    Console.WriteLine("Hello");
}
```

## Step-by-Step Explanation
1. A function is declared with a name.
2. The function body contains code.
3. The function is called when needed.

## Best Practices
- Give functions meaningful names.
- Keep each function focused on one task.

## Common Mistakes
- Forgetting to call the function.
- Writing one very large function instead of smaller ones.
