# Break Statement

## Concept Explanation
The `break` statement is used to stop the execution of a loop or `switch` statement immediately.

## Real-World Analogy
Think of `break` like pressing an emergency stop button on a machine.

## Syntax
```csharp
if (condition)
{
    break;
}
```

## Step-by-Step Explanation
1. The loop starts running.
2. A condition is checked.
3. When the condition becomes true, `break` stops the loop.

## Best Practices
- Use `break` only when stopping early makes the code clearer.
- Make the stop condition easy to understand.

## Common Mistakes
- Using `break` without understanding which loop it exits.
- Overusing `break` and making logic harder to read.
