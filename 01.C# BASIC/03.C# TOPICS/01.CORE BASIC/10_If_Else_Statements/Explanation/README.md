# If-Else Statements

## Concept Explanation
The `if-else` statement lets a program choose between two paths based on a condition.

If the condition is true, the `if` block runs.
If the condition is false, the `else` block runs.

## Real-World Analogy
Think of it like an ID check at an entrance:

- If age is 18 or above, enter
- Else, do not enter

## Syntax
```csharp
if (condition)
{
    // true block
}
else
{
    // false block
}
```

## Step-by-Step Explanation
1. A condition is evaluated.
2. If it is `true`, the `if` block runs.
3. Otherwise, the `else` block runs.

## Best Practices
- Keep conditions short and clear.
- Use braces even for small blocks.
- Make the output meaningful.

## Common Mistakes
- Forgetting braces.
- Using `=` instead of `==` when comparing values.
- Writing conditions that are difficult to read.
